# Configuração do GitHub

## Opção 1: Usando GitHub CLI (recomendado)

Se você tem o GitHub CLI instalado:

```bash
cd /home/gabv/Projetos/osrs-good-to-flip
gh repo create osrs-good-to-flip --public --source=. --remote=origin --push
```

## Opção 2: Criar manualmente no GitHub

1. Acesse https://github.com/new
2. Crie um novo repositório chamado `osrs-good-to-flip`
3. **NÃO** inicialize com README, .gitignore ou license (já temos)
4. Execute os comandos abaixo:

```bash
cd /home/gabv/Projetos/osrs-good-to-flip
git remote add origin https://github.com/SEU_USUARIO/osrs-good-to-flip.git
git branch -M main
git push -u origin main
```

## Opção 3: Usando SSH

Se você prefere usar SSH:

```bash
cd /home/gabv/Projetos/osrs-good-to-flip
git remote add origin git@github.com:SEU_USUARIO/osrs-good-to-flip.git
git branch -M main
git push -u origin main
```

## Verificar configuração do Git

Se quiser alterar o nome/email do commit:

```bash
git config user.name "Seu Nome"
git config user.email "seu-email@example.com"
```

Ou globalmente:

```bash
git config --global user.name "Seu Nome"
git config --global user.email "seu-email@example.com"
```
